# GameControllerBehavior

An app to demonstrate differences in Game Controller behavior between iOS and visionOS.

To use:

1. Pair a game controller with either an iOS or a visionOS device.
2. Tap the _Start Session_ button to initiate a FaceTime call to another device with a different Apple ID.
3. Accept the Group Activity on the other device.
4. Tap _Connect Game Controller_ on the device paired with the game controller.
5. Operate the Game controller. Events should appear on the other device.
